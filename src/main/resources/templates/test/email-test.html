EcoVivaShop - Prueba de Correos

📧 Sistema de Prueba de Correos Electrónicos

Herramienta para probar los diferentes tipos de correos de EcoVivaShop

Correo de Bienvenida - Usuario

Envía un correo de bienvenida para nuevos usuarios registrados.

Correo de Bienvenida - Admin

Envía un correo de bienvenida para nuevos administradores.

Confirmación de Pedido

Envía confirmación de pedido realizado.

Información del Sistema

  • Configurar spring.mail.username
  • Configurar spring.mail.password
  • Usar contraseña de aplicación de Gmail
🎯 Templates Disponibles:
  • email/bienvenida.html
